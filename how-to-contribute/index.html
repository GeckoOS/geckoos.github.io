How to Contribute

There are several ways you can contribute to the OS or the website:

  • Fix a typo or add better wording
  • Improve Documentation
  • Add a new feature
  • Improve or optimize code
  • Clean up code
  • Format code and comments
  • Tell others about the project

You can apply your contributions to the main project by forking our GitHub repository, making a change, and submitting a pull request.

If you contribute at all in any way to the project, please add yourself to the contributors.json file in the repository for this website and submit a pull request.
